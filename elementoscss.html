RGB

Podemos establecer el valor de un color mezclando diferentes intensidades de los tres colores primarios. sintaxis: rgb(red, green, blue)

RGBA

Podemos definir la transparencia de un color usando la función rgba, esta función es idéntica a la función rgb, pero con un parámetro adicional denominado opacidad, para especificar la transparencia. Ejm: elemento{ color: rgba(37, 150, 190, 1); }

HEXADECIMAL

Podemos utilizar un número hexadecimal para definir un color. Es una alternativa a la función rgb, es decir permite definir un color como la mezcla de intensidades de los colores primarios Ejm:elemento{ color: #numero-hexadecimal; }

HSL

En este modelo utilizamos conceptos que son más fáciles de comprender. En CSS para poder trabajar con este modelo tenemos la siguiente opción Ejm:elemento{ color: hsl(h, s, l); }

SELECTORES

SELECTOR UNIVERSAL (*)

Permiten seleccionar elementos del documento a los que queremos aplicar estilos. sintaxis: *{ declaraciones CSS }

SELECTORES DE TIPO (p)

Permite seleccionar elementos usando el nombre del elemento. nombre_elemento { declaraciones CSS }

SELECTOR DE CLASE (.foo)

Permite seleccionar elementos en función de los valores del atributo class de los elementos. Para que los elementos sean seleccionados, su atributo class debe incluir un valor que coincida con el valor del selector. .valor_atributo { declaraciones CSS }

SELECTORES DE ID (#foo)

Permite seleccionar un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, su atributo id debe coincidir exactamente con el valor dado en el selector. #valor_atributo { declaraciones CSS }

SELECTORES DE ATRIBUTO (a[foo])

Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo. selector[attr] { } selector[attr=value] {} selector[attr~=value] {}

PSEUDO- ELEMENTOS

Seleccionan parte de un elemento en lugar del elemento mismo. Los pseudo-elementos se añaden a los selectores y permiten añadir estilos a una parte concreta de los elementos seleccionados. Por ejemplo, el pseudo elemento ::first-line selecciona solo la primera línea del elemento especificado por el selector. Sintaxis: selector::pseudo-elemento { … }

FLEX BOX

flex-start

- No existe distancia entre el lado inicial del eje principal y su elemento adyacente.

- No existe distancia entre los elementos contiguos de una línea

- Si el contenedor flexible es de una sola línea y es demasiado pequeño para contener

todos los elementos, los elementos flexibles se desbordan en dirección del eje principal.

flex-end

- No existe distancia entre el lado final del eje principal y su elemento adyacente.

- No existe distancia entre los elementos contiguos de una línea.

- Si el contenedor flexible es de una sola linea y es demasiado pequeño para contener

todos los elementos, los elementos flexibles se desbordan en dirección contraria del eje principal.

center

- No existe espacio entre los elementos adyacentes de una línea.

- La distancia entre los lados del eje principal y sus elementos adyacentes es la misma.

- Si el contenedor flexible es de una sola línea y es demasiado pequeño para contener

todos los elementos, los elementos flexibles se desbordan por igual en ambas direcciones.

space-between

- No existe espacio entre los lados del eje principal y sus elementos adyacentes.

- La distancia entre los elementos adyacentes de una linea es la misma.

- Si solo existe un elemento flexible en la línea, este se coloca pegado al lado inicial del eje principal.

- Si el contenedor flexible es de una sola línea y es demasiado pequeño para contener todos los elementos, los elementos flexibles se desbordan en dirección del eje principal.

space-around

- La distancia entre elementos adyacentes de una línea es la misma.

- La distancia entre los lados del eje principal y sus elementos adyacentes es la mitad de la distancia que existe entre elementos adyacentes.

- Si solo existe un elemento flexible en la línea, este se coloca al medio.

- Si el contenedor flexible es demasiado pequeño para contener todos los elementos y es de una sola linea, los elementos flexibles se desbordan por igual en ambas direcciones.
